# Hypothesis Generator

Generate concrete testing hypotheses for media buying campaigns.

## When to use
- User is stuck and needs fresh test ideas
- User says "new hypotheses", "ideas for testing", "what to test next"

## Instructions

Collect from user:
- Vertical being run
- Platform
- Current metrics: CTR%, CVR%, CPA$
- What was already tested without success
- What worked well
- Constraints (budget, policies, etc.)

Generate **5 concrete, actionable hypotheses**. For each include:
- **What to test** exactly
- **Why it could work** (reasoning)
- **How to measure** (primary metric + threshold)
- **Effort level** (low/medium/high)

## Rules
- Never suggest something already tested (check context)
- Each hypothesis must have an isolatable variable
- Must be specific: "Test UGC video vs static image" not "try different creatives"
- Prioritize low-effort high-impact ideas first
- Consider the vertical's compliance restrictions
