# Prompt Original — Análisis de Test Completado

```
Actuá como un media buyer senior analizando los resultados de un A/B test.

CONTEXTO DEL TEST:
- Qué se testeó: [describir qué variable se cambió]
- Hipótesis original: [por qué creíamos que iba a funcionar]
- Plataforma: [Meta / TikTok / Google / Native]
- Vertical: [auto insurance / etc]
- Duración: [X] días
- Budget total: $[X]

RESULTADOS:
Variante A ([descripción]):
- Spend: $
- CTR: %
- CVR: %
- CPA: $
- [otras métricas relevantes]:

Variante B ([descripción]):
- Spend: $
- CTR: %
- CVR: %
- CPA: $
- [otras métricas relevantes]:

BASELINE ANTERIOR: [métrica de referencia si existe]

Respondé con exactamente esta estructura:
1. CONCLUSIÓN (1 oración)
2. GANADOR y por qué
3. ¿Es el resultado estadísticamente significativo? (sí/no + razonamiento simple)
4. APRENDIZAJE TRANSFERIBLE (qué nos dice esto más allá de este test puntual)
5. NEXT ACTION recomendada (específica y accionable)
6. NUEVA HIPÓTESIS sugerida basada en este resultado
```
